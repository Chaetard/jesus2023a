Ejemplo de HTML5 y CSS

Este es un ejemplo que muestra algunas de las funciones básicas, intermedias y avanzadas de HTML5 y CSS.

Funciones básicas

Las funciones básicas incluyen el uso de etiquetas HTML como <p> para párrafos, <a> para enlaces y <img> para imágenes.

Funciones intermedias

Las funciones intermedias incluyen el uso de estilos CSS para dar formato a los elementos HTML, así como el diseño de una rejilla de diseño flexible utilizando display: flex y flex-wrap.
